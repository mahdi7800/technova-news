== Description ==
TechNova Pro یک قالب پیشرفته وردپرسی با امکانات کامل برای ساخت وب‌سایت‌های حرفه‌ای است. این قالب با قابلیت‌های سفارشی‌سازی بالا و ابزارهای حرفه‌ای طراحی شده تا نیازهای کاربران حرفه‌ای را به‌طور کامل پوشش دهد.

== Features ==
* ابزارک‌های اختصاصی برای سایدبار و فوتر
* مگامنو با پشتیبانی از چند سطح منو
* فوتر اختصاصی با تنظیمات دلخواه
* ویجت‌های قابل تنظیم برای بخش‌های مختلف
* پنل تنظیمات اختصاصی قالب
* سیستم ثبت‌نام در صورت انتخاب با پنل پیامکی
* با وب سرویس ملی پیامک
* سیستم SMTP برای ارسال ایمیل‌های امن
* متاباکس‌های سفارشی برای پست‌ها
* نمایش مطالب مرتبط و پربحث‌ترین مطالب
* خبرنامه (Newsletter) با قابلیت اتصال به سرویس‌های ایمیلی
* سازگار با:
  - Bootstrap 5
  - UIKit
  - jQuery Toast
  - Tiny Slider
  - Font Awesome
  - Sticky JS
* طراحی واکنش‌گرا و پشتیبانی از زبان‌های راست‌چین (RTL)

== Installation ==
1. فایل قالب را در مسیر `/wp-content/themes/technova-pro/` قرار دهید.
2. وارد پیشخوان وردپرس شوید.
3. به بخش "نمایش » پوسته‌ها" بروید.
4. قالب "TechNova Pro" را فعال کنید.

== لایسنس ==
برای فعال‌سازی کامل امکانات قالب، لطفاً در پنل تنظیمات قالب، کد لایسنس خود را وارد نمایید. در صورت نداشتن کد، با پشتیبانی تماس بگیرید.
